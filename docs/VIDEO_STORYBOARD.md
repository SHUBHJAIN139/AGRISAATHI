## Production Notes

### Assets Needed
1. **Lakshmi portrait** — Generate via Imagen or use Creative Commons Indian farmer stock photo
2. **Leaf disease photos** — PlantVillage dataset (public domain)
3. **Architecture diagram** — Export from Mermaid in the app
4. **Antigravity screen recording** — Capture during actual build session
5. **Phone mockup** — Record PWA in Chrome DevTools mobile emulation
6. **Background music** — Royalty-free Indian instrumental (e.g., from YouTube Audio Library)

### Recording Equipment
- Screen: OBS Studio (free, 1080p @ 30fps)
- Narration: Any USB microphone, quiet room
- Editing: DaVinci Resolve (free) or Kdenlive

### Accessibility
- Closed captions in English and Hindi
- Audio descriptions for visual elements
- High contrast text overlays (WCAG AA minimum)
